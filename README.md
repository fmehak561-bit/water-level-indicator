# Water Level Indicator

## Project Description
This project is a simple Water Level Indicator designed using Tinkercad. It indicates the water level in a tank using multiple level indicators (LEDs).

## Components Used
- Arduino Uno
- LEDs
- Resistors
- Jumper Wires
- Water Level Sensor/Probes
- Breadboard


## Features
- Detects different water levels.
- Displays water level using LEDs.
- Simple and easy-to-understand design.
